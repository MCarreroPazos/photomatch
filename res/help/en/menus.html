File

Nuevo proyecto

Crea un nuevo proyecto vacio. En caso de que exista un proyecto o modelo abierto se cierran.

Método abreviado de teclado: Ctrl+N

Abrir proyecto

Se despliega una ventana que permite cargar un proyecto existente.

Método abreviado de teclado: Ctrl+O


Proyectos recientes

Se muestra un listado de los diez últimos proyectos abiertos permitiendo el acceso rápido a los mismos.

Desde el ménu Borrar historial se elimina el listado de ficheros recientes.

Guardar

Se guardan los cambios en el proyecto.

Método abreviado de teclado: Ctrl+S

Guardar como

Permite guardar con otro nombre el proyecto.

Método abreviado de teclado: Ctrl+Shift+S

Cerrar proyecto

Cierra el proyecto actual.

Salir

Cierra CrashMap.

Método abreviado de teclado: Ctrl+F4

View

Desde el menú Ver se activan o desactivan los paneles acoplables y las barras de herramientas.

Paneles acoplables

  • Proyecto: Abre el panel Árbol de proyecto. Si el panel esta abierto aparece resaltado con una marca de chequeo.
  • Propiedades: Abre el Propiedades. Si el panel esta abierto aparece resaltado con una marca de chequeo.
  • Miniaturas: Abre el Panel miniaturas. Si el panel esta abierto aparece resaltado con una marca de chequeo.
