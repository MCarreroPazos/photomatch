File

The File Menu provides commands for creating, opening and saving PhotoMatch projects and exporting features y matches.


New Project

Create a new empty project. In case there is an open project, it closes.


Keyboard shortcut: Ctrl+N

Open Project

Open an existing project.

Keyboard shortcut: Ctrl+O

Recent Projects

A list of the latest open projects is shown allowing quick access to them. In Settings → General it is possible to modify the number of projects saved in the history.

Clear History deletes the list of recent files.

Save Project

Saves the current project

Keyboard shortcut: Ctrl+S

Save Project As...

Save Project As... allows you to save a copy of an open project to a different name or location

Keyboard shortcut: Ctrl+Shift+S

Export

The Export menu allows you to export keypoints and matches in various formats.

Close Project

Close the current project.

Exit

Close PhotoMatch. If there are any unsaved changes to your project, PhotoMatch will ask if you want to save them.

Keyboard shortcut: Ctrl+F4
