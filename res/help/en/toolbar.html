Toolbars

The toolbars provide quick access to some functions of PhotoMatch.

  • General
  • Image Viewer
  • Tools
  • Quality Control

General Toolbar

the General toolbar provides access to the following items from the File menu

The General Toolbar provide access to the following items from the File menu:

  • New Project
  • Open Project
  • Save
  • Save As...

Image Viewer Toolbar

  • Zoom In
  • Zoom Out
  • Zoom 1:1
  • Zoom Extend

Tools Toolbar

  • Add Images
  • Add Session
  • Chage active session
  • Image preprocessing
  • Feature Extraction
  • Feature Matching

Quality Control Toolbar

  • ROC Curves
  • DET Curves
